### Prerequisites

* nodejs
* mongodb

1. Install server dependencies
    ```bash
    $ cd server
    $ npm install
    ```
2. Install client dependencies
    ```bash
    $ cd client
    $ npm install
    ```

## Run the app
1. Start the server
    ```bash
    $ cd server
    $ npm start
    ```
2. Start the client
    ```bash
    $ cd client
    $ yarn start
    ```
3. Browse to `http://localhost:3000/`
